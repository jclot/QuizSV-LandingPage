In this section you will find all the information about the application.

How to use this application specifically?

When you have installed the application two buttons will appear saying (Start | Show Instructions) if you press the start button you will go to the video recording where you can edit everything and start recording the screen, if you press the show instructions button then you will find the instructions of how to start in the application.

How do you see the application directly? where is each thing? how do you modify it?

When you start the application and press start, you will find different buttons and a black screen. The buttons are (choose screen | stop screen | start recording | stop recording | choose extension file) when you press choose screen a small menu will open where you can choose from the available screens to show them, if you press stop screen will stop presenting the screen and return to the start, when you press start recording and stop recording simply what to do is to start recording the screen, but of course first you have to put the screen and finally if you press choose extension file you will find a menu where you can choose the different extensions to save the video.

How do I give permission to the application to record and display the screen?

If you are a mac user then this information will be very useful, when you download the application you have to give some permissions to the application to do some things, when you try to record or display the screen will ask you to go to system preferences then Security & Privacy and you will see a menu with 4 buttons simply go to the one that says general and below all give the button that is called open anyways, or you appear then cancel the alert that the system launched or try to record or display the screen again.
